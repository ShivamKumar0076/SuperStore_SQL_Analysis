# 📊 SuperStore SQL Analysis

## 📌 Project Overview
This project analyzes the Global SuperStore dataset using PostgreSQL. The goal is to solve real-world business problems using SQL and generate meaningful business insights.

---

## 🛠 Tools Used
- PostgreSQL
- pgAdmin 4
- SQL

---

## 📂 Dataset
- Global SuperStore Dataset
- Rows: 51,290

---

## 📚 SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- HAVING
- Aggregate Functions
- CASE WHEN
- Joins
- Self Join
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- LAG()
- LEAD()
- FIRST_VALUE()
- LAST_VALUE()

---

## 📈 Business Questions Solved

- Top 5 customers by total sales
- Top profitable products in each category
- Lowest sales customer in each category
- Customers with sales above market average
- Previous and next customer sales
- Sales difference from previous order
- First and last sale of every customer
- Customer sales contribution (%)
- Profit Margin Analysis
- Business Summary using SQL
